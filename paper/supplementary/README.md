# Supplementary Material

This directory holds supplementary material accompanying the paper
*Post-Quantum Secure API Communication: Architectures and Migration
Strategies for Quantum-Resilient Systems*.

## Contents

- `notation-and-abbreviations.md` — Full expansion of acronyms and
  symbols used in the paper, provided for readers unfamiliar with
  NIST PQC or IETF TLS terminology.
